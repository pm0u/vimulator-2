  • Help! I'm stuck in Vim! How do I quit?

    • This very popular post on stackexchange is a great resource

  • General Vim Resources

    • The Vim Wiki - a wealth of information, primarily formatted as 'tips.' They also have a great tutorial page
    • Vim Stack Exchange - a place to ask questions and search/browse other question/answers related to vim
    • Vim on the Arch Wiki - the arch wiki is a great resource for anything linux. Mostly focused on configuration/customization (some is arch specific)
    • Vim man page - primarily technical information. also available from your terminal with $man vim
    • Learning the Vi Editor - a wiki about Vi (predecessor to Vim) and Vim

  • Other Interactive Tutorials

    • Open Vim - a similar open source resource that aims to re-create vim in the browser.
    • Vim Adventures - a cute dungeon style game that uses vim shortcuts to control characters and complete the game.
    • VimGenius - a thorough interactive vim tutorial designed to be used alongside vimtutor
    • Vimtutor - an interactive vim tutorial available from your terminal, jsut type $vimtutor ! (assuming you have vim installed)

  • Customizing Vim

    • Vim Awesome - a huge list of plugins for Vim that greatly increase functionality. These require a plugin manager (see below)
    • Solarized color scheme - the color scheme used for this website -- a well thought out color scheme intended to increase readability while minimizing strain
    • Fira Mono - font used on this page, designed for looking at code. (Google fonts has many great free monospace fonts)

  • Plugin Managers

    • Vundle - the most popular plugin manager
    • Vim-plug - A minimalist Vim plugin manager.
    • Pathogen - Manage your 'runtimepath' with ease. In practical terms, pathogen.vim makes it super easy to install plugins and runtime files in their own private directories.
    • Dein.vim - designed to replace NeoBundle, a full featured plugin manager
